## Slack이란?

Slack은 **업무용 메신저**입니다. 카카오톡과 비슷해 보이지만, 업무에 최적화된 구조를 갖추고 있습니다.

| 카카오톡 | Slack |
|---------|-------|
| 개인/그룹 톡 중심 | 채널(주제별 방) 중심 |
| 파일 보관 기간 짧음 | 영구 검색 가능 |
| 업무/개인 혼용 | 업무 전용 |
| 스레드 없음 | 스레드로 대화 정리 |
| 알림 제어 어려움 | 정교한 알림 설정 |
| 봇/자동화 없음 | 앱 연동, 자동화 가능 |

## 앱 설치

**Desktop (권장)**
- Windows: https://slack.com/downloads/windows
- Mac: https://slack.com/downloads/mac
- 또는 App Store / Microsoft Store 검색 "Slack"

**Mobile (보조)**
- iOS: App Store → "Slack"
- Android: Google Play → "Slack"

> 세션 실습은 반드시 **Desktop 앱**으로 참여하세요. 웹 브라우저도 가능하지만 기능이 제한됩니다.

## 워크스페이스 참가

1. 관리자(운영지원본부)로부터 초대 이메일 또는 초대 링크 수령
2. 이메일의 "Join Now" 버튼 클릭 또는 링크 접속
3. 대학 이메일(@chu.ac.kr)로 계정 생성
4. 이름 설정: **한국어 실명** (예: 이영준) — 닉네임 사용 금지
5. 프로필 사진 설정 권장

## 기본 기능

### 채널 (Channel)
주제별로 나뉜 대화 공간. `#` 기호로 시작합니다.

| 채널 | 용도 |
|------|------|
| `#anchor-workshop` | 워크숍 공지 및 Q&A |
| `#general` | 전체 공지 |
| `#random` | 자유 대화 |
| `#ai-tips` | AI 활용 팁 공유 |

### 메시지
- 채널 하단 입력창에 메시지 작성
- `Enter` = 전송, `Shift+Enter` = 줄바꿈

### 스레드 (Thread)
메시지에 달린 **답글 모음**. 채널을 깔끔하게 유지합니다.
- 메시지에 마우스 올리기 → "Reply in thread" 클릭
- 관련 대화는 반드시 스레드로!

### 멘션 (@)
특정 사람에게 알림 보내기.
- `@이영준` → 해당 인물에게 알림
- `@channel` → 채널 전체 (자제 요망)
- `@here` → 현재 온라인 멤버

### 이모지 반응
메시지에 이모지로 반응. 짧은 답장 대신 사용합니다.
- ✅ = 확인했습니다
- 👀 = 보고 있습니다
- 🙏 = 감사합니다
- 👍 = 좋아요/동의

### 파일 공유
메시지창의 📎 아이콘 또는 드래그앤드롭으로 파일 첨부.
Google Drive 파일은 링크로 공유 (Drive 파일 직접 첨부 지양).

## 검색 활용법

상단 검색창 (`Cmd+K` / `Ctrl+K`) 또는 `Cmd+F` / `Ctrl+F`

| 검색 필터 | 예시 |
|----------|------|
| 특정 채널 | `in:#anchor-workshop 예산` |
| 특정 사람 | `from:이영준` |
| 파일만 | `has:file` |
| 날짜 | `after:2026-04-01` |

## 카카오톡 vs Slack: 실전 비교

| 상황 | 카카오톡 방식 | Slack 방식 |
|------|-------------|-----------|
| 공지 사항 | 단톡방에 메시지 | `#general` 채널에 게시 |
| 파일 공유 | 파일 전송 (기간 지나면 사라짐) | 채널에 업로드 (영구 보관) |
| 긴 대화 | 계속 이어서 작성 | 스레드로 묶기 |
| 빠른 확인 | "ㅇㅋ", "확인" 텍스트 | ✅ 이모지 반응 |
| 밤 메시지 | 즉시 알림 | 수신자가 알림 시간 설정 |
