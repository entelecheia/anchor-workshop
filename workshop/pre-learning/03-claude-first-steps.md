# Claude와 첫 대화

> 사전학습 3/4 | 소요 시간: 약 40분

---

## Claude란?

Claude는 Anthropic이 만든 **AI 어시스턴트**입니다. 텍스트를 읽고, 쓰고, 분석하고, 번역하고, 코드를 짜는 등 다양한 지적 작업을 도울 수 있습니다.

우리 사업단에서는 주로 이런 용도로 씁니다:
- 보고서 초안 작성 및 개조식 변환
- 회의록 정리
- 공문·이메일 초안
- 예산 데이터 해석 및 분석 요약
- 한영/영한 번역

---

## 계정 만들기

1. **claude.ai** 접속
2. "Sign up" 클릭
3. Google 계정(대학 이메일 권장) 또는 이메일로 가입
4. 이름·직책 입력 후 완료
5. 무료 플랜으로 시작 (월 사용량 제한 있음)

> 사업단에서 Claude Pro/Max 구독 지원을 검토 중입니다. 우선 무료 플랜으로 시작하세요.

## Claude Desktop 설치

Desktop 앱을 설치하면 웹보다 더 빠르고 편리하게 사용할 수 있습니다.

- **Mac**: https://claude.ai/download → macOS 버전 다운로드
- **Windows**: 같은 링크 → Windows 버전 다운로드
- 설치 후 claude.ai 계정으로 로그인

## 첫 대화 해보기

설치 후 다음 3가지 질문을 직접 해보세요.

**질문 1: 자기소개 요청**
```
안녕! 넌 뭘 잘하니? 대학 행정 업무에 어떻게 도움을 줄 수 있어?
```

**질문 2: 간단한 변환 요청**
```
다음 문장을 공식 보고서 개조식으로 바꿔줘:
"이번 달에 행사를 했는데 학생들이 500명 정도 참여했고 반응이 좋았어요."
```

**질문 3: 요약 요청**
```
다음 내용을 3줄로 요약해줘:
[아무 보도자료나 붙여넣기]
```

결과가 맘에 들지 않으면 "더 짧게", "더 격식체로", "항목을 추가해서" 같이 이어서 요청하면 됩니다. 이것이 **프롬프트 이터레이션**입니다.

## Claude vs ChatGPT: 간단 비교

| 항목 | Claude | ChatGPT |
|------|--------|---------|
| 만든 곳 | Anthropic | OpenAI |
| 장문 처리 | 매우 강함 (200K 토큰) | 좋음 |
| 안전성 | 보수적, 신중 | 다양 |
| 한국어 | 좋음 | 좋음 |
| 코드 생성 | 우수 | 우수 |
| 웹 검색 | Claude.ai에서 가능 | 가능 |
| 무료 플랜 | 있음 (제한적) | 있음 (제한적) |

> 이 프로그램에서는 Claude를 주로 사용합니다. 회사 정책상 업무 데이터를 AI에 입력할 때는 **개인정보·미공개 예산 정보 제외** 원칙을 지켜주세요.

## 할 수 있는 것

- 긴 문서 읽고 요약하기
- 초안 작성 (보고서, 이메일, 공문, 기획서)
- 표·데이터 해석 및 분석 코멘트
- 한국어 ↔ 영어 번역
- 아이디어 브레인스토밍
- 복잡한 개념 쉽게 설명

## 주의할 점

- **사실 확인 필수**: Claude가 틀린 정보를 자신있게 말할 수 있습니다 (환각, hallucination)
- **개인정보 입력 금지**: 학생 개인정보, 직원 급여, 미공개 예산 내역
- **최종 책임은 사람**: AI 초안은 반드시 검토 후 사용
- **저작권**: AI 생성 콘텐츠를 그대로 외부 게시 시 출처 표기 고려
