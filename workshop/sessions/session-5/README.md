## 실습 내용

### 실습 1: 공문 번역
`exercises/official-letter-template.md`의 한국어 공문 샘플을 영어로 번역:

```
다음 한국 대학 공문을 공식 영문 서한으로 번역해줘.
- 대학명: Cheju Halla University
- 사업명: RISE Program (Regional Innovation Support for Education)
- 격식체 영어 사용
- 날짜 형식: April 13, 2026

[공문 내용 붙여넣기]
```

### 실습 2: 공문 초안 작성
```
다음 상황에 맞는 공문 초안을 작성해줘:
- 발신: 제주한라대학교 RISE 사업단
- 수신: [협력기관명]
- 목적: MOU 체결 일정 조율 요청
- 일시: 2026년 5월 말 희망
- 형식: 표준 공문 형식 (문서번호, 수신·발신, 제목, 내용, 붙임)
```

### 실습 3: 영문 이메일 초안
```
다음 상황의 영문 이메일을 작성해줘:
- 발신자: Young Joon Lee, Associate Professor, Cheju Halla University
- 수신자: International partner (Pix4D)
- 목적: Asia Hub Training Center 협력 일정 확인
- 톤: 전문적, 우호적
- 길이: 150-200단어
```
