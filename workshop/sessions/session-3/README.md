## 핵심 개념: 개조식 작성 원칙

한국 공식 문서의 개조식 원칙:

- **명사형 종결**: "~함", "~임", "~예정"
- **항목 번호**: 1. → 가. → 1) → 가) 순서
- **수치 명시**: 추상적 표현 → 구체적 수치로
- **간결성**: 한 항목에 한 내용, 불필요한 수식어 제거

**나쁜 예**: 이번에 행사를 진행했는데 학생들이 많이 와서 성공적으로 마무리되었습니다.
**좋은 예**: 행사 개최 완료 (참가자 312명, 목표 대비 124%)

## 실습 내용

### 실습 1: 개조식 변환 마스터
```
다음 역할을 맡아줘: 대학 행정 문서 전문 편집자
아래 비격식 보고 내용을 공식 개조식 보고서로 변환해줘.
- 항목 번호 체계 사용 (1. → 가. → 1))
- 명사형 종결어 사용
- 수치는 그대로 유지하되 단위 명확히
- 각 항목 20자 이내

[내용 붙여넣기]
```

### 실습 2: 대상별 요약
같은 분기 보고서를 두 가지 버전으로 요약:
- 버전 A: 학교장 보고용 (5줄, 핵심 수치 중심)
- 버전 B: 팀 내부 공유용 (항목별 상세)

### 실습 3: 이터레이션 연습
결과가 마음에 들지 않을 때 사용하는 후속 프롬프트:
- "이 항목을 더 구체적으로 써줘"
- "수치가 빠진 항목을 찾아서 [미확인]으로 표시해줘"
- "전체 분량을 30% 줄여줘"
