## 핵심 개념: 재사용 가능한 프롬프트 설계

**나쁜 프롬프트** (일회용):
```
4월 행사 보고서를 개조식으로 바꿔줘. 참가자 312명이고 만족도 4.2점이야.
```

**좋은 프롬프트** (재사용 가능한 템플릿):
```
대학 행정 보고서 편집자로서, 아래 비격식 행사 결과를 공식 개조식 보고서로 변환해줘.

행사명: {행사명}
일시: {일시}
참가자: {참가 인원}명
주요 성과: {성과 내용}
특이 사항: {특이 사항}

출력 형식:
1. 행사 개요 (3항목 이내)
2. 주요 성과 (수치 중심)
3. 향후 계획 (1-2항목)
각 항목 20자 이내, 명사형 종결
```

## 실습 내용

### 실습 1: 나만의 프롬프트 설계
1. 가장 자주 하는 반복 텍스트 작업 1개 선택
2. 프롬프트 4요소(역할·맥락·지시·형식) 포함하여 템플릿 작성
3. `{placeholder}` 형태로 변수 처리
4. 3회 테스트 후 완성

### 실습 2: 프롬프트 라이브러리 기여
완성된 프롬프트를 `prompt-library/` 폴더 형식에 맞게 작성:
- 용도, 프롬프트, 사용 예시, 팁 포함
- Slack `#anchor-workshop`에 공유 (팀 리뷰 후 라이브러리에 추가)
