## 실습 내용

### 실습 1: 샘플 예산 분석
`exercises/budget-sample.csv` 내용을 복사해 Claude에 붙여넣기 후:

```
다음은 RISE 사업단 1분기 예산 집행 현황입니다.
각 사업부별로:
1. 집행률(%) 계산
2. 집행률이 낮은(70% 미만) 항목 파악
3. 전체 집행률 및 잔액 합계
4. 담당자에게 전달할 한 줄 코멘트 작성

[CSV 붙여넣기]
```

### 실습 2: 보고 코멘트 생성
```
위 분석 결과를 바탕으로 부단장 보고용 2단락 요약을 작성해줘.
- 전체 집행 현황 (1단락)
- 주의가 필요한 항목 및 권고사항 (1단락)
- 개조식, 명사형 종결
```

### 실습 3: 시각화 설명 요청
```
위 데이터를 막대차트로 시각화한다면 어떤 차트가 적합할지 설명해줘.
Google Sheets에서 만드는 방법도 알려줘.
```
