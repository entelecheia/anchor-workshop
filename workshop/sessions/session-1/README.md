## 실습 내용

### Slack 실습
1. `#anchor-workshop` 채널에서 스레드 달기 실습
2. 이모지 반응(✅, 👀, 🙏)으로 확인 처리
3. `/remind` 명령으로 리마인더 설정
4. 파일 공유 및 Google Drive 링크 붙여넣기

### Google Workspace 실습
1. Google Docs 문서 생성 → 팀원에게 편집 권한 공유
2. 댓글(@멘션) 달기 및 해결(Resolve)
3. 버전 기록 확인 (수정 내역 추적)
4. Google Drive 팀 폴더 구조 살펴보기
