# 회의록 자동 구조화

## 용도

메모 형태로 작성된 회의 기록을 정식 회의록 형식으로 변환합니다. 액션 아이템(담당자·기한 포함)을 자동 추출하여 후속 관리를 용이하게 합니다.

---

## 프롬프트

```
당신은 대학 행정 회의록 전문 작성자입니다.

아래 회의 메모를 정식 회의록으로 변환해주세요.

**출력 형식**:

## 회의 정보
- 일시: {날짜 및 시간}
- 장소: {장소}
- 참석자: {참석자 목록}
- 불참자: {불참자 및 사유}
- 작성자: {작성자}

## 안건별 논의 내용

### 1. {안건명}
- 논의 내용 요약 (개조식)
- 결정 사항: (있는 경우)

### 2. {안건명}
...

## 결정 사항 요약
| 번호 | 결정 내용 | 근거 |
|------|----------|------|

## 액션 아이템
| 번호 | 할 일 | 담당자 | 기한 | 상태 |
|------|-------|--------|------|------|
| 1 | | | | 진행중 |

## 다음 회의
- 일시: {다음 회의 일정}
- 주요 안건: (예상되는 경우)

**변환 규칙**:
- 불명확한 담당자나 기한은 [확인필요]로 표시
- 메모에 없는 내용은 추가하지 않음
- 개조식, 명사형 종결

**회의 메모**:
{메모 붙여넣기}
```

---

## 사용 예시

**입력**:
```
4/3 목 2시 회의실
참석: 이영준, 문현주, 조인범
예산 73% 집행됨. 코랩 납기 5월로 밀렸다고 장용준 교수가 얘기함.
런케이션 공사 4/17까지 완료 약속. 정아영이 매주 확인하기로.
다음 회의 4/17.
```

**출력**:
```
## 회의 정보
- 일시: 2026년 4월 3일(목) 14:00
- 장소: 회의실
- 참석자: 이영준, 문현주, 조인범
- 불참자: [확인필요]
- 작성자: [확인필요]

## 안건별 논의 내용

### 1. 예산 집행 현황
- 전체 집행률 73% 보고
- R&D 코랩 구축 장비 납기 5월로 지연 확인

### 2. 런케이션 공사 진행 현황
- 업체 공사 완료 목표: 4월 17일
- 매주 현장 확인 체계 수립

## 액션 아이템
| 번호 | 할 일 | 담당자 | 기한 | 상태 |
|------|-------|--------|------|------|
| 1 | 런케이션 공사 현장 주간 확인 | 정아영 | 매주 | 진행중 |
| 2 | 코랩 장비 납기 확정 일정 공유 | 장용준 | [확인필요] | 진행중 |

## 다음 회의
- 일시: 2026년 4월 17일(목) 14:00
```

---

## 팁

- **녹취 텍스트도 가능**: 자동 녹취 텍스트(Clova Note, Plaud AI 등)를 그대로 붙여넣어도 정리됩니다
- **긴 회의**: 회의 내용이 길면 "안건 1~3만 먼저 정리해줘"처럼 나눠서 처리
- **액션 아이템 추출만**: "위 메모에서 액션 아이템만 표로 뽑아줘"로 단독 사용도 가능
- **후속 공유**: 완성된 회의록을 Slack 스레드에 붙여넣으면 참석자 전원이 확인 가능
