# 기본 보고서 개조식 변환

## 용도

비격식 구어체 또는 서술형으로 작성된 업무 내용을 한국 공식 문서 표준인 개조식 보고서로 변환합니다. 분기 보고서, 행사 결과 보고, 사업 진행 현황 보고 등에 활용합니다.

---

## 프롬프트

```
당신은 대학 행정 문서 전문 편집자입니다.

아래 비격식 업무 내용을 공식 개조식 보고서로 변환해주세요.

**변환 규칙**:
- 항목 번호 체계: 1. → 가. → 1) → 가) 순서 사용
- 종결어: 명사형 (예: ~함, ~임, ~예정, ~완료)
- 수치: 원문 수치를 그대로 유지하되 단위 명확히 표기
- 항목 길이: 한 항목 20자 이내
- 내용 없는 항목 추가 금지

**보고서 구조**:
1. {섹션1 제목} (예: 추진 현황)
2. {섹션2 제목} (예: 주요 성과)
3. {섹션3 제목} (예: 향후 계획)

**변환할 내용**:
{비격식 텍스트 붙여넣기}
```

---

## 사용 예시

**입력**:
```
이번 달에 AI 융합전공 수업을 다 개강했어요. 5개 트랙에 학생이 187명 왔고
작년보다 23% 늘었습니다. 수업 만족도는 4.2점이에요.
```

**출력**:
```
1. 추진 현황
  가. AI 융합전공 5개 트랙 전체 개강 완료
  나. 수강생 187명 등록 (전년 동기 대비 23% 증가)

2. 주요 성과
  가. 수업 만족도 4.2점/5.0점 달성

3. 향후 계획
  가. 2분기 중간 성과 점검 예정
```

---

## 팁

- **섹션 이름을 구체적으로** 지정할수록 구조가 좋아집니다 (예: "추진 현황 / 주요 성과 / 과제 및 계획")
- **수치가 많을 때**: "수치가 있는 항목은 반드시 포함하고 없는 항목만 [미확인]으로 표시해줘" 추가
- **분량 조절**: "전체 항목 수를 10개 이내로 줄여줘" 또는 "각 섹션 최소 3항목씩"
- **이터레이션**: 결과가 마음에 들지 않으면 "이 항목을 더 구체적으로 써줘" 또는 "이 표현을 더 격식체로 바꿔줘"
