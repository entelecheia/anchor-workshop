# 공식 문서 번역 (한국어 ↔ 영어)

## 용도

대학 공문, MOU 서한, 협력 이메일 등 공식 문서의 한영·영한 번역에 사용합니다. 일반 번역기와 달리 대학 행정 맥락과 격식체를 유지합니다.

---

## 프롬프트 (한국어 → 영어)

```
당신은 한국 대학 공식 문서 전문 번역가입니다.

아래 한국어 {문서 유형}을 공식 영문 서한으로 번역해주세요.

**번역 규칙**:
- 대학명: Cheju Halla University (제주한라대학교)
- 사업명: RISE Program (앵커/RISE 사업)
- 직함: Associate Professor (부교수), Deputy Director (부단장)
- 날짜 형식: April 13, 2026
- 톤: 격식체, 전문적이되 우호적
- 문화적 표현 (예: "귀사의 무궁한 발전을 기원합니다") → 영어 관용 표현으로 자연스럽게 변환

**번역할 내용**:
{한국어 문서 붙여넣기}
```

---

## 프롬프트 (영어 → 한국어)

```
당신은 한국 대학 행정 문서 전문 번역가입니다.

아래 영문 {문서 유형}을 한국어 공식 공문 형식으로 번역해주세요.

**번역 규칙**:
- 격식체 한국어 사용 (~하였습니다, ~드립니다)
- 대학명: 제주한라대학교
- 날짜 형식: 2026년 4월 13일
- 외래어 고유명사(기업명, 프로그램명)는 원문 유지 + 한국어 병기
- 영어 행정 용어는 괄호 안에 원문 표기 (예: MOU(업무협약))

**번역할 내용**:
{영어 문서 붙여넣기}
```

---

## 사용 예시

**입력 (한→영)**:
```
제목: 산학협력 협약 체결 협조 요청
...이에 귀사와의 협력 강화를 위해 업무협약(MOU) 체결을 요청드리오니...
```

**출력**:
```
Subject: Request for Cooperation in MOU Signing
...In this regard, we would like to formally propose the signing of a
Memorandum of Understanding (MOU) to strengthen our partnership...
```

---

## 팁

- **문서 유형을 명확히**: "공문", "이메일", "MOU 서한", "보도자료" 등을 명시하면 형식이 달라집니다
- **고유명사 목록 제공**: 번역 전에 "다음 고유명사는 아래처럼 번역해줘: [목록]" 을 추가하면 일관성이 높아집니다
- **톤 조정**: "더 격식체로", "좀 더 친근하게", "짧게 요약해서" 등 후속 요청 가능
- **검토 필수**: AI 번역은 반드시 담당자가 검토 후 발송하세요. 특히 직함, 날짜, 수치는 원문과 대조 확인
