## A. 업무자동화 (Task Automation)

**설명**: 기존에 수작업으로 반복하던 업무를 Claude 프롬프트 또는 워크플로우로 자동화한 사례

**대상 업무 예시**:
- 매주 반복되는 보고서 초안 작성
- 회의록 → 액션 아이템 추출 자동화
- 공문·이메일 템플릿 기반 초안 생성
- 예산 집행 현황 → 요약 코멘트 자동 생성
- 행사 결과 → SNS 홍보 콘텐츠 변환

**평가 포인트**:
- 자동화 전/후 소요 시간 비교 (정량 지표)
- 재현 가능성 (다른 사람이 따라 쓸 수 있는가)
- 실제 업무에 적용한 증거

## B. AI 어시스턴트 (AI Assistant)

**설명**: 특정 업무 영역에 특화된 맞춤형 프롬프트 시스템 또는 Claude 활용 워크플로우 구축

**대상 사례 예시**:
- "RISE 공문 작성 도우미" — 상황 입력하면 공문 초안 완성
- "예산 분석 어시스턴트" — 데이터 붙여넣으면 보고 코멘트 생성
- "회의록 자동 정리기" — 메모 입력 → 정식 회의록 출력
- "영문 파트너 이메일 생성기" — 상황 설명 → 영문 이메일 초안

**평가 포인트**:
- 프롬프트 품질 및 재현성
- 업무 적합성 (실제 사용 가능한가)
- 사용 편의성 (팀원 누구나 쓸 수 있는가)

## C. 협업혁신 (Collaboration Innovation)

**설명**: Slack + Claude + Google Workspace를 연계하여 팀 협업 방식을 개선한 사례

**대상 사례 예시**:
- Slack 채널 운영 체계 설계 + 실제 도입 사례
- 공유 문서 + Claude 활용 동시 편집 워크플로우
- 팀 공유 프롬프트 라이브러리 구축 및 운영
- Claude로 회의 전 자료 준비 → 회의 중 메모 → 회의 후 정리 연결

**평가 포인트**:
- 팀 전체의 협업 효율 개선도
- 지속 운영 가능성 (일회성이 아닌 체계)
- 팀원 만족도
