## 경진대회 안내

**기간**: 5/15(목) 킥오프 → 5/22(목) 제출 마감 → 5/28(수) 데모데이

| 카테고리 | 설명 |
|----------|------|
| A. 업무자동화 | 반복 업무를 Claude로 자동화한 사례 |
| B. AI 어시스턴트 | 맞춤형 프롬프트/워크플로우 구축 |
| C. 협업혁신 | Slack + Claude 연계 팀 협업 개선 |

**시상**: 대상(50만원) / 최우수(30만원) / 우수(20만원) / 인기상(10만원) + Claude 구독권

## 참여 방법

1. **사전학습**: `workshop/pre-learning/` 폴더 순서대로 읽고 환경 설정 완료
2. **체크리스트**: `workshop/pre-learning/checklist.md` 항목 모두 완료 후 Slack 인증
3. **세션 참여**: 날짜에 맞춰 오프라인 세션 참석 (zoom 병행 예정)
4. **경진대회**: 팀(2-4인) 구성 후 `challenge/templates/submission-readme.md` 참고하여 제출

## 저장소 구조

```
anchor-workshop/
├── workshop/
│   ├── pre-learning/   사전학습 자료 (4개)
│   ├── sessions/       세션별 자료 (6개)
│   └── exercises/      실습 파일 (샘플 데이터, 초안 등)
├── challenge/
│   ├── rules/          경진대회 규정
│   ├── templates/      제출 양식
│   ├── submissions/    팀 제출물 (PR로 추가)
│   └── results/        결과 발표
├── prompt-library/     업무별 프롬프트 모음 (기여 환영)
└── guidelines/         AI 사용 정책, Slack 문화 가이드
```
